## How To Use

```bash
# Clone this repository
$ git clone https://github.com/54hg0220/SpringBoot-React.git
```
**Database** MySQL8
```bash
# Create employee table in your local db
$ create table ems
```

**MainEntry** of the frontend

```bash

$ npm install
npm start
```
